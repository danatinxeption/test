The problems

warning For the HTML/CSS exercises in this project, please make sure to not use any CSS frameworks like Boostrap, Tailwind, Foundation, etc. For the javascript test, please don't use any component libraries.

Project Setup

  1. Create a NextJS app with TypeScript.
  2. Set up SASS project in the app.
  3. Create four routes:
    1. / - Application homepage.
    2. /center-a-div - Page for exercise one.
    3. /table - Page for exercise two.
    4. /javascript - Page for exercise three.
  4. Bonus point for using dynamic routing to handle some or all four pages above.

Exercises One - Center a DIV

  1. Create a div that takes up the entire viewport width and height.
  2. Give it a solid background color.
  3. Add a child element with "Center This" text.
  4. Center the element above in the middle of the page, both horizontal and vertically.
  5. Use as little html/css as possible.
  6. Bonus point for completing this exercise with multiple methods.

Exercise Two - Table layout

  1. Create a three by four layout.
    1. On viewport > 1280px, the layout is 3 x 4.
    2. On viewport > 650px and < 1279px, the layout changes to 2 x 6.
    3. On viewport < 650px, the layout changes to 1 x 12.
  2. Around the layout, there should be a 50px gap to the edge of the screen.
  3. The cell width in all the viewport sizes. The cell height be at least 100px.
  4. The space between the cells should be 50px, both horizontal and vertical.
  5. Add a tooltip at the top right corner of each element.
    1. The tooltips should only be visible on hover of the cell it attaches to.

Exercise Three - Javascript

Getting the data

  1. Use this free data api endpoint documentation: https://docs.irail.be/
  2. Get all the train stations from Belgium Public Transport api endpoint.

Using the data

  1. Create a table.
  2. The table columns should be dynamically generated using the train stations data.
  3. The columns should be Name, Latitude, Longitude.
  4. Add a note column as the last column in the table.
  5. In the table row, display the train station data in the table cell. For the note column, add an input field and ability to type notes for each row.
  6. Bonus points:
    1. Creating a link to google maps using the latitude and longitude information and make the name a link, also open the link in a new tab.
    2. Add search functionality.
    3. Add sort functionality.
